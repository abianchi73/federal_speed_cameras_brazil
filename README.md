
# Federal Speed Cameras and Road Safety in Brazil

Replication package for the paper:

Author(s): Andre Giavina Bianchi, Vinicius Lobosco, Lycia Lima, Ciro Biderman
Journal status: Working paper

## Structure
- data/: processed data
- code/: estimation scripts
- output/: generated figures/tables
- docs/: manuscript and appendix

## Replication

Restore packages:

renv::restore()

Run:

source("master_run.R")

## Software
R version 4.x
Packages recorded in renv.lock
